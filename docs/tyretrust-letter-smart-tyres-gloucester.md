# TyreTrust letter — Smart Tyres Gloucester (Jiri)

Per-prospect direct-mail letter for the Tier-1 **pre-built** approach (site built
before posting). Speed-first / walk-in positioning = TyreTrust's **"The Site"**
plan (£9.99/mo, pay-what-you-think build). Deliberately does **not** sell the
booking engine, but leaves the door open to the booking version in case the
walk-in read is wrong.

- **Brand:** TyreTrust throughout, `nick@tyretrust.uk` (assumes the domain +
  mailbox exist before posting — pre-flight blocker).
- **Owner:** Jiri (confirmed via Companies House — Ltd company director).
- **No prospect email needed:** this is posted mail; the response path is the QR
  → the site → Nick's contact details on the page.

## Pre-flight checklist (lock before it prints)

1. **QR target = the PLAIN site** `/smart-tyres-gloucester/` — NOT the booking
   variant `/smart-tyres-gloucester-booking/`. The booking version contradicts
   the "no diary" pitch. (The booking variant is only for the "if I've misjudged
   it" follow-up.)
2. **`tyretrust.uk` domain + mailbox live** so the letterhead/sign-off address
   works.
3. **Price wording:** letter and landing pages now match — "pay what you think
   it's worth; as a guide £150–£300; pay less if money's tight or you don't think
   it's worth that." Keep them in sync if either changes.
4. **Envelope:** stamped, hand-addressed, no window (per the campaign's
   highest-leverage rule in `docs/tyretrust-mailshot-copy.md`).
